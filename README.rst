INSTALL
--------

.. code-block:: bash

    $ pip install python-masscan

USAGE
--------

.. code-block:: python

 import masscan


  mas = masscan.PortScanner()
  mas.scan('192.168.1.1', ports='22,1900', arguments='--max-rate 1000')
  print(mas.scan_result)
